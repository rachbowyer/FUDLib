FUDLibrary

FUDLibrary is a set of over 60 User Defined Functions for the Interbase database. It includes numerical, string and date functions.


Version and status of this release

This release is version 0.5. It should be viewed as a 'Test' or 'Beta Release' and should only be used in production environments with extreme care.


Where to get new versions?

The latest version is available from my website.

Mer System's UDFLib

FUDLibrary, although an entirely separate and unrelated product is functionally compatible with UDFLib. UDFLib is proprietary closed source library available from MER Systems Inc.


Platforms

The only supported platforms are
Windows NT 4.0
Interbase v6

It should work with Windows 9X and Windows 2000, but I have not tested it on those platforms. For Unix please read the build notes.
